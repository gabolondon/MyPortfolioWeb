Social wall at Circuloten App build using React-Native, Django, expo-location and TensorFlow

In a dynamic collaboration, the product owner envisioned a captivating 'Social Section,' and that's where my journey began. With just a two-week timeline, I teamed up with another developer to transform this vision into a reality. Our tool of choice was React Native, which allowed us to craft an engaging user experience. We meticulously designed screens and components, ensuring seamless navigation and data transfer between them. Expo-location was instrumental in fetching precise user locations, a crucial element for our upcoming API requests. As we advanced with our front-end work, I pivoted towards building the back-end infrastructure using Django Rest Framework APIs. The data science team then furnished me with a TensorFlow-powered script, which worked its magic by recommending users with shared interests and pinpointing proximities using location data. Once the pieces were in place, we seamlessly integrated our APIs with the React front-end. The result? A captivating Social Section that seamlessly connects users and sparks meaningful interactions. This project showcased my ability to orchestrate a harmonious symphony between front-end and back-end development, delivering an exceptional user experience.
